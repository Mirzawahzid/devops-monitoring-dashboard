# DevOpsHub 🚀

Cloud operations dashboard built with React, Node.js, SQL Server, Docker Compose, and JWT authentication.

## 🔥 Features

- Secure JWT Login Authentication
- Real-time Metrics Dashboard
- KPI Cards & Charts
- Dockerized Full Stack Deployment
- SQL Server Database
- REST API Backend
- Responsive Premium UI

## 🛠 Tech Stack

- React + Vite
- Node.js + Express
- SQL Server
- Docker + Docker Compose
- JWT Auth

## 📦 Run Locally

```bash
docker compose up --build
